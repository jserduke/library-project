# Library System Project
## How to Start Using It
1. Clone our repository locally.
2. Open our project in Eclipse.
3. Right-click on LibrarySystem/src/server/LibraryServer.java and select "Run as Java Application" to run the server.
4. Check the server's IP address in the console output, replace the placeholder IP address in line 20 of LibrarySystem/src/client/Client.java with it, and save it.
5. Right-click on LibrarySystem/src/client/Client.java and select "Run as Java Application" to run the client.
6. The GUI will appear and you can start using our application!
## How to Run the Tests
1. Right-click on LibrarySystem/src/test/allTestsSuite.java and select "Run as JUnit Test".
2. The JUnit tab will open next to the file explorer and all of the application's unit tests will run!
